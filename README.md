# CWC Clovis Theme

A warm Celtic/monastic WordPress theme for the Community of the Coworkers of Christ Clovis NM Cell.

## Features
- Sacramental, prayer-shaped community focus
- Member portal integration
- Daily prayers and liturgies
- Community events and news
- Prayer requests and giving

## Installation
1. Place this theme in `wp-content/themes/`
2. Activate in WordPress admin
3. Configure menus and settings
